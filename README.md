# French Fluency Trainer

An AI-powered web application that helps French learners improve listening comprehension using personalized practice generated from the sentences they already understand.

## Overview

French Fluency Trainer bridges the gap between memorizing French and understanding spoken French in real conversations.

Instead of presenting random listening exercises, the application uses a learner's existing Anki sentence cards to determine their current level. AI then generates new, comprehensible sentences of similar difficulty, allowing users to practice listening with content tailored to what they already know.

The goal is to provide an endless source of personalized listening practice without spending time searching for level-appropriate videos or podcasts.

## Motivation

Traditional language-learning tools each solve a different problem:

- **Duolingo** introduces vocabulary and grammar.
- **Anki** reinforces long-term memory through spaced repetition.

However, many learners still struggle to understand spoken French.

French Fluency Trainer was built to transform existing knowledge into meaningful listening practice.

## Features

- Import French sentences from Anki exports
- Analyze the learner's existing knowledge
- Generate new AI-powered practice sentences
- Listen to generated French using text-to-speech
- Type what you hear to reinforce listening skills
- Compare your answer with the correct sentence and translation
- Practice through a clean, distraction-free interface
